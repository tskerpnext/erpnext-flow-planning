# ERPNext 銷售訂單 — 自動選倉說明書

**建立日期：** 2026-04-17  
**更新日期：** 2026-04-21（物料組更名為簡體中文）  
**適用版本：** Frappe v16 / ERPNext v16

---

## 功能說明

當使用者在銷售訂單新增**第一筆品項**時，系統自動根據該品項的**物料組（Item Group）**，設定整張訂單的**預設發貨倉（Set Warehouse）**。

| 物料組 | 自動填入倉庫 |
|---|---|
| 电缆产品 | 成品仓 - SKT |
| 加工产品 | 成品仓 - SKT |
| Raw Material | 物料仓 - SKT |

---

## 實作方式

使用 **Client Script**，DocType = `Sales Order`，監聽子表 `Sales Order Item` 的 `item_code` 事件。

### 腳本名稱
`Sales Order-自動選倉`

### 腳本內容

```javascript
frappe.ui.form.on('Sales Order Item', {
    item_code: function(frm, cdt, cdn) {
        let row = locals[cdt][cdn];
        let items_len = frm.doc.items ? frm.doc.items.length : 0;

        // 只在第一筆品項被選擇時執行
        if (items_len === 1 && row.item_code) {
            frappe.db.get_value('Item', { name: row.item_code }, 'item_group')
                .then(r => {
                    let item_group = (r && r.message) ? r.message.item_group : null;
                    let target_warehouse = '';

                    if (item_group === '电缆产品') {
                        target_warehouse = '成品仓 - SKT';
                    } else if (item_group === '加工产品') {
                        target_warehouse = '成品仓 - SKT';
                    } else if (item_group === 'Raw Material') {
                        target_warehouse = '物料仓 - SKT';
                    }

                    if (target_warehouse) {
                        setTimeout(() => {
                            frm.set_value('set_warehouse', target_warehouse);
                            frm.refresh_field('set_warehouse');
                        }, 500);
                    }
                });
        }
    }
});
```

---

## 技術說明

### 為什麼只判斷第一筆品項

`set_warehouse` 是整張訂單的全域倉庫設定，只需根據第一筆品項決定一次。  
判斷條件：`frm.doc.items.length === 1`（子表只有一行時才執行）。

### 為什麼用 `setTimeout(..., 500)`

ERPNext 的 `item_code` 事件觸發後，系統內部會執行品項資料回填（名稱、UOM、單價等）。加入 500ms 延遲確保我們的 `set_value` 在系統回填完成後才執行，避免被覆蓋。

### API 回應格式

`frappe.db.get_value()` 的回傳格式為：

```json
{ "message": { "item_group": "电缆产品" } }
```

必須用 `r.message.item_group` 取值。

## 新增物料組規則

在腳本的條件判斷區塊加入：

```javascript
} else if (item_group === '新物料組名稱') {
    target_warehouse = '新倉庫 - SKT';
}
```
