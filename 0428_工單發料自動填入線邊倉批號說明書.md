# 工單發料自動填入線邊倉批號說明書

建立日期：2026-04-28
適用版本：ERPNext V16
狀態：✅ 已部署測試通過

---

## 一、背景說明

工單發料（Material Transfer for Manufacture）時，來源倉為 `线边仓 - SKT` 的物料需要指定批號才能提交。

若每次手動查詢批號庫存再逐一填入，操作繁瑣且容易出錯。尤其當單一批號庫存不足，需要跨多批號湊量時，更需要分拆成多行手動處理。

本功能透過 Client Script，在發料單建立時自動查詢線邊倉批號庫存，依優先順序填入批號與數量，並在庫存不足時提供明確提示。

---

## 二、批號分配規則

| 優先順序 | 說明 |
|---------|------|
| **第一優先** | 批號 `000`（預設批號，固定優先取用） |
| **其後依序** | 其他有庫存的批號，依批號名稱字母排序 |

### 各情況處理結果

| 情況 | 結果 |
|------|------|
| 批號 `000` 庫存足夠 | 一行，batch=000，qty=應領量 |
| `000` 不足，其他批號可補足 | 多行：000 取盡 + 其他批號依序補足 |
| 所有批號合計仍不足 | 最後一行 qty 超出可用量 → **無法提交**，提示使用者補料 |
| `000` 無庫存、其他批號足夠 | 直接從有庫存的批號開始（不顯示空的 000 行） |
| 完全無庫存 | 一行 batch=000，qty=應領量 → **無法提交** |

---

## 三、觸發方式

| 觸發 | 時機 |
|------|------|
| **自動** | 手動設定物料號（`item_code`）或來源倉改為 `线边仓 - SKT` 時，自動執行 |
| **手動按鈕** | 點擊表單工具列的「**自動填入線邊倉批號**」，適用 BOM 帶入後批量處理 |

> BOM 自動帶入物料時，欄位變更事件可能不會觸發，建議使用**手動按鈕**統一處理。

---

## 四、執行邏輯

```
1. 篩選物料行：s_warehouse = 线边仓 - SKT 且非成品行
2. 呼叫 erpnext.stock.doctype.batch.batch.get_batch_qty
   取得該物料在線邊倉所有批號的當前庫存
3. 批號分配演算法：
   a. 優先從批號 000 取用：take = min(stock_000, required)
   b. 若仍有剩餘，依批號名稱排序取其他有庫存批號
   c. 若全部批號合計仍不足：最後一個批號 qty += 剩餘量
      （該行 qty 超出可用庫存 → ERPNext 提交時報錯）
4. 以 frm.clear_table() + frm.add_child() 重建物料行（避免鬼魂列）
5. 未受影響的物料行（非線邊倉）原樣保留
```

---

## 五、注意事項

- 僅處理 `Material Transfer for Manufacture` 類型的 Stock Entry
- 僅對 `s_warehouse = 线边仓 - SKT` 的物料行生效，其他來源倉不受影響
- 使用 ERPNext 內建的 `get_batch_qty` 查詢，只回傳**當前庫存 > 0** 的批號
- 批號欄位變更後，ERPNext 會自動重新計算估值與金額
- 不自動處理序號（Serial No），序號仍需手動填寫

## 七、Client Script 程式碼

**DocType：** `Stock Entry`　**類型：** Form

```javascript
const LINESIDE_WH = '线边仓 - SKT';

// ── 表單層：工具列按鈕 ────────────────────────────────────
frappe.ui.form.on('Stock Entry', {
    refresh: function(frm) {
        if (frm.doc.stock_entry_type === 'Material Transfer for Manufacture' &&
            frm.doc.docstatus === 0) {
            frm.add_custom_button(__('自動填入線邊倉批號'), function() {
                fill_all_lineside_batches(frm);
            });
        }
    }
});

// ── 物料行層：欄位變更自動觸發 ───────────────────────────
let _filling = false;

frappe.ui.form.on('Stock Entry Detail', {
    item_code: function(frm, cdt, cdn) {
        let row = locals[cdt][cdn];
        if (!_filling &&
            frm.doc.stock_entry_type === 'Material Transfer for Manufacture' &&
            row.s_warehouse === LINESIDE_WH && row.item_code && flt(row.qty) > 0) {
            setTimeout(() => fill_row_batches(frm, cdt, cdn), 500);
        }
    },
    s_warehouse: function(frm, cdt, cdn) {
        let row = locals[cdt][cdn];
        if (!_filling &&
            frm.doc.stock_entry_type === 'Material Transfer for Manufacture' &&
            row.s_warehouse === LINESIDE_WH && row.item_code && flt(row.qty) > 0) {
            fill_row_batches(frm, cdt, cdn);
        }
    }
});

// ── 單列批號填入 ─────────────────────────────────────────
function fill_row_batches(frm, cdt, cdn) {
    if (_filling) return;
    let row = locals[cdt][cdn];
    fetch_batch_allocs(row.item_code, LINESIDE_WH, flt(row.qty)).then(allocs => {
        _filling = true;
        frappe.model.set_value(cdt, cdn, 'batch_no', allocs[0].batch_no);
        frappe.model.set_value(cdt, cdn, 'qty', allocs[0].qty);

        for (let i = 1; i < allocs.length; i++) {
            let nr = frm.add_child('items');
            let skip = new Set(['name','idx','__islocal','__unsaved',
                                'doctype','parent','parentfield','parenttype',
                                'batch_no','qty','transfer_qty']);
            Object.keys(row).forEach(k => { if (!skip.has(k)) nr[k] = row[k]; });
            nr.batch_no = allocs[i].batch_no;
            nr.qty = allocs[i].qty;
            nr.transfer_qty = flt(allocs[i].qty) * flt(row.conversion_factor || 1);
        }
        frm.refresh_field('items');
        _filling = false;
    });
}

// ── 全表批號填入（按鈕觸發）─────────────────────────────
function fill_all_lineside_batches(frm) {
    let targets = (frm.doc.items || []).filter(r =>
        r.s_warehouse === LINESIDE_WH && r.item_code && !r.is_finished_item
    );
    if (!targets.length) {
        frappe.show_alert({ message: '沒有來源倉為線邊倉的物料行', indicator: 'orange' });
        return;
    }

    Promise.all(
        targets.map(row =>
            fetch_batch_allocs(row.item_code, LINESIDE_WH, flt(row.qty))
                .then(allocs => ({ row, allocs }))
        )
    ).then(results => {
        let repl = {};
        results.forEach(({ row, allocs }) => { repl[row.name] = { row, allocs }; });

        let orig = JSON.parse(JSON.stringify(frm.doc.items || []));
        let skip = new Set(['name','idx','__islocal','__unsaved',
                            'doctype','parent','parentfield','parenttype',
                            'batch_no','qty','transfer_qty']);
        let keep = new Set(['name','idx','__islocal','__unsaved',
                            'doctype','parent','parentfield','parenttype']);

        frm.clear_table('items');
        orig.forEach(item => {
            if (repl[item.name]) {
                repl[item.name].allocs.forEach(alloc => {
                    let nr = frm.add_child('items');
                    Object.keys(item).forEach(k => { if (!skip.has(k)) nr[k] = item[k]; });
                    nr.batch_no = alloc.batch_no;
                    nr.qty = alloc.qty;
                    nr.transfer_qty = flt(alloc.qty) * flt(item.conversion_factor || 1);
                });
            } else {
                let nr = frm.add_child('items');
                Object.keys(item).forEach(k => { if (!keep.has(k)) nr[k] = item[k]; });
            }
        });

        frm.refresh_field('items');
        frm.dirty();
        frappe.show_alert({ message: '線邊倉批號已自動填入', indicator: 'green' });
    });
}

// ── 批號分配演算法 ────────────────────────────────────────
function fetch_batch_allocs(item_code, warehouse, required_qty) {
    return new Promise(resolve => {
        frappe.call({
            method: 'erpnext.stock.doctype.batch.batch.get_batch_qty',
            args: { item_code: item_code, warehouse: warehouse },
            callback: r => resolve(allocate_batches(r.message || [], required_qty))
        });
    });
}

function allocate_batches(batches, required_qty) {
    required_qty = flt(required_qty);

    // 建立庫存對照表（get_batch_qty 只回傳 qty > 0 的批號）
    let stock = {};
    (batches || []).forEach(b => { stock[b.batch_no] = flt(b.qty); });

    let remaining = required_qty;

    // Step 1：優先取批號 000
    let take_000 = Math.min(stock['000'] || 0, remaining);
    remaining -= take_000;

    // Step 2：其他批號依名稱排序補足
    let other_allocs = [];
    let other_keys = Object.keys(stock).filter(k => k !== '000').sort();
    for (let bn of other_keys) {
        if (remaining <= 0) break;
        let take = Math.min(stock[bn], remaining);
        other_allocs.push({ batch_no: bn, qty: take });
        remaining -= take;
    }

    // Step 3：仍不足 → 剩餘量加到最後一個批號（超出庫存，無法提交）
    if (remaining > 0) {
        if (other_allocs.length > 0) {
            other_allocs[other_allocs.length - 1].qty += remaining;
        } else {
            take_000 += remaining;
        }
    }

    // 組合結果（000 無庫存且有其他批號時不顯示 000 空行）
    let result = [];
    if (take_000 > 0 || other_allocs.length === 0) {
        result.push({ batch_no: '000', qty: take_000 });
    }
    return result.concat(other_allocs);
}
```

## 八、已知限制

- BOM 自動帶入物料時欄位事件可能不觸發，需手動點擊按鈕補執行
- `get_batch_qty` 只回傳當前庫存 > 0 的批號；批號存在但庫存為 0 時不會出現在選項中
- 批號名稱排序以字母順序為準，若需自訂優先順序須修改 `allocate_batches` 函式
