# 物料組與倉庫中文化變更說明

日期：2026-04-21

---

## 一、物料組（Item Group）更名

| 原名稱（英文）   | 新名稱（簡體中文） |
|----------------|----------------|
| HM Products    | 加工产品         |
| Cable Products | 电缆产品         |

**操作方式：** 使用 `frappe.rename_doc` 在 ERPNext 後端執行更名，資料庫中所有關聯欄位自動同步。

---

倉庫名稱已為簡體中文，本次無更名，列出供參考：

| 倉庫名稱（含公司後綴）   | 倉庫名稱         |

## 三、受影響的自訂腳本

物料組更名後，以下 Client Script 中硬寫的物料組名稱需同步修改。**本次已一併更新完畢。**

### 1. BOM自动规划发料仓

**位置：** ERPNext → 自訂 → Client Script → BOM自动规划发料仓
**觸發時機：** BOM 中新增品項（`BOM Item.item_code` 改變時）
**邏輯說明：** 根據品項的物料組自動填入「發料倉庫」

| 物料組     | 自動填入倉庫    |
|-----------|--------------|
| 电缆产品   | 线边仓 - SKT  |
| Raw Material | 物料仓 - SKT |

**本次變更：**
```
舊：if (item_group === 'Cable Products')
新：if (item_group === '电缆产品')
```

---

### 2. Sales Order-自動選倉

**位置：** ERPNext → 自訂 → Client Script → Sales Order-自動選倉
**觸發時機：** 銷售訂單新增第一筆品項時（`Sales Order Item.item_code` 改變）
**邏輯說明：** 根據品項的物料組自動填入訂單層級的「預設倉庫」（set_warehouse）

| 物料組     | 自動填入倉庫    |
|-----------|--------------|
| 电缆产品   | 成品仓 - SKT  |
| 加工产品   | 成品仓 - SKT  |
| Raw Material | 物料仓 - SKT |

**本次變更：**
```
舊：if (item_group === 'Cable Products')
新：if (item_group === '电缆产品')

舊：} else if (item_group === 'HM Products')
新：} else if (item_group === '加工产品')
```

---

## 四、注意事項

- **Raw Material** 物料組名稱本次未更改，仍維持英文，上述腳本中對應部分無需修改。
- 若未來再新增自訂腳本有用到物料組名稱，請使用簡體中文（`加工产品`、`电缆产品`）。
- 倉庫名稱目前已統一為簡體中文，新增腳本請直接使用中文倉庫名稱（含公司後綴 `- SKT`）。
